Backend Payment Gateway

Backend_Payment_Gateway is a Node.js-based secure transaction system with a MySQL database (locally hosted, but adaptable to online hosting). It verifies users before generating a bcrypt-encrypted token for payment, ensuring authenticity. Once the transaction is completed, the token is destroyed, and payment details (user ID, amount, etc.) are securely stored. While actual UPI payments require legal verification, this system provides a secure and scalable payment framework ready for real-world integration.
